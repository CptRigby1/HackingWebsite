Lazy Script

Commands To Install LazyScript

apt-get install git

git clone https://github.com/arismelachroinos/lscript.git

cd lscript

chmod +x install.sh

./install.sh

Open another terminal and type "l" then press enter.

Commands To uninstall LazyScript

cd /root/lscript

./uninstall.sh

rmdir -r /root/lscript
